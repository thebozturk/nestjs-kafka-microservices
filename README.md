<h1>About</h1>
<p>This repository has been created to understand the logic of Nest.js microservices with Kafka.<p>

<h2>Technologies</h2>

<h3>Installation</h3>

 `$ git clone https://github.com/thebozturk/nestjs-kafka-microservices.git` 
 
 <br>
